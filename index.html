About Us

Don't know how to ensure the security of multiple system files? Wanna know if they are stored according to the CIS benchmarks? Our project helps you to know whether all your files are secured properly according to CIS benchmarks by running audit scripts remotely. It checks and tells if there are any files which don't follow this benchmark.

Project Overview

This project checks whether all the files in your systems are secured according to CIS benchmarks or not. It runs an audit script in your system and its outputs are checked in the remotely located system.
